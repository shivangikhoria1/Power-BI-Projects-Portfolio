# BankSim Fraud Detection Dashboard

## 📊 Project Overview

This project analyzes transactional data from the BankSim dataset to identify potential fraud patterns using Power BI.

## 🔍 Key Features

* Fraud vs Non-Fraud Analysis
* Transaction Patterns
* Risk Segmentation
* KPI Metrics for Fraud Detection

## 🛠 Tools Used

* Power BI
* DAX
* Power Query

## 📁 Files Included

* Power BI Dashboard (.pbix)
* Dataset (.csv)
* Dashboard Screenshots

## 💡 Insights

* Identified high-risk transaction patterns
* Highlighted regions with higher fraud activity
* Provided actionable insights for fraud prevention
