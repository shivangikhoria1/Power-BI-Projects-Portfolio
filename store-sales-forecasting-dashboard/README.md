# Store Sales Forecasting Dashboard with Time-Based Insights

## 📊 Project Overview
This project analyzes historical store sales data and predicts future sales trends using forecasting techniques.

It helps businesses:
- Understand sales patterns over time
- Identify peak and low sales periods
- Make data-driven inventory and business decisions

## 📁 Files Included
- Store_Sales_Forecasting.pbix (Power BI File)
- Store.csv (Dataset)
- Dashboard Screenshots

## 📌 Key Insights
- Sales show seasonal patterns with peak periods
- Certain categories contribute more to revenue
- Profit varies significantly across regions

## ⚠️ Problems Identified
- Sales drop in off-season periods
- Some products generate low or negative profit
- Demand fluctuation creates inventory challenges

## 💡 Recommendations
- Increase stock before forecasted peak periods
- Focus on high-performing categories
- Reduce or optimize low-profit products

## 📈 Forecast Strategy
- Sales expected to increase in upcoming period
- Businesses should prepare inventory accordingly
- Plan marketing campaigns based on predicted trends

## 📈 Tools Used
- Power BI (Forecasting feature)
- Excel / CSV Dataset
