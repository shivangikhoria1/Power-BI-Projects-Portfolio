# Healthcare Analytics Dashboard - Hospital Performance & Patient Insights

## 📊 Project Overview
This project analyzes healthcare data to understand:
- Patient admissions & discharge trends
- Hospital revenue analysis
- Department performance
- Bed occupancy and resource utilization

## 📁 Files Included
- Healthcare_Analytics.pbix (Power BI File)
- SQL Queries Screenshots
- Dashboard Screenshots

## 📌 Key Insights
- Certain departments have higher patient load
- Peak admission periods identified
- Revenue is driven by specific treatments/services

## ⚠️ Problems Identified
- Uneven patient distribution across departments
- Resource underutilization in some areas
- High load during peak periods

## 💡 Recommendations
- Optimize staff allocation
- Improve resource management
- Plan for peak demand periods

## 📈 Tools Used
- Power BI
- SQL  Server Management Studio
