# HR Analytics Dashboard - Attrition & Workforce Insights

## 📊 Project Overview
This project analyzes employee data to understand:
- Employee Attrition
- Workforce Trends
- Department-wise Analysis
- Salary & Job Role Insights

## 📁 Files Included
- HR_Analytics.pbix (Power BI File)
- HR_Data.csv (Dataset)
- Dashboard Screenshots

## 📌 Key Insights
- High attrition observed in specific departments
- Younger employees show higher attrition rate
- Salary and job role impact employee retention

## ⚠️ Problems Identified
- High turnover in certain roles
- Low employee satisfaction in some departments

## 💡 Recommendations
- Improve employee engagement programs
- Offer better salary structure
- Focus on retention strategies

## 📈 Tools Used
- Power BI
- Excel / CSV Dataset
