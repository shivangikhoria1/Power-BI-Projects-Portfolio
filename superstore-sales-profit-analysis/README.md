# Superstore Sales & Profit Analysis Dashboard

## 📊 Project Overview
This project analyzes Superstore sales data to understand revenue, profit, and business performance across different regions, categories, and time periods.

## 📁 Files Included
- Superstore_Analysis.pbix (Power BI File)
- Superstore.csv (Dataset)
- Dashboard Screenshots

## 📌 Key Insights
- Some regions generate high sales but low profit
- Certain product categories are more profitable
- Discounts significantly impact profit margins

## ⚠️ Problems Identified
- High discounts leading to profit loss
- Loss-making products in specific categories
- Regional imbalance in performance

## 💡 Recommendations
- Optimize discount strategy
- Focus on high-profit products
- Improve performance in low-performing regions

## 📈 Business Insights
- Identify strong vs weak areas
- Improve pricing and discount decisions
- Increase profitability through data-driven strategy

## 📈 Tools Used
- Power BI
- Excel / CSV Dataset
